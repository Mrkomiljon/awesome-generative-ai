# Contributing Guide

Thanks for helping improve this list. Please keep contributions clear, consistent, and easy to review.

## What to Add
- Open-source or freely accessible resources
- Short, factual descriptions
- Links to official repos, papers, or demos

## Where to Add It
- Speech: `stt-models.md`, `tts.md`, `voice-cloning.md`, `emotion-recognition.md`
- Vision: `text-to-image.md`, `talking-head.md`
- Agents: `ai-agents.md`
- APIs and models: `genai-apis.md`, `transformers.md`
- MCP servers: `mcp.md`

## Formatting Rules
- Use plain ASCII (avoid emoji and special characters)
- Keep descriptions short and consistent
- Follow the existing section order
- One resource per bullet or table row
- Do not add promotional language

## Suggested Entry Format
```
- [Project Name](https://github.com/org/repo) - One-line description
```

## Submitting Changes
1. Fork the repo
2. Create a branch
3. Make your edits
4. Open a pull request with a short summary
